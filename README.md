## 🚀 โปรเจกต์จบ ปวช.3 — เว็ปไซต์หมู่บ้านเจซีการ์เดนวิลล์

**📌 ระบบนี้คืออะไร?**

ระบบบริหารจัดการหมู่บ้านผ่านเว็บไซต์ ที่ผสานเทคโนโลยี AI และสมองกลฝังตัวอัจฉริยะ เพื่ออำนวยความสะดวกแก่ลูกบ้านและนิติบุคคล เช่น ระบบแจ้งปัญหาอัตโนมัติ ระบบจองสถานที่ ระบบชำระค่าส่วนกลางออนไลน์ และระบบรักษาความปลอดภัยอัจฉริยะผ่าน IoT และ AI ทำให้หมู่บ้านมี
ความปลอดภัย ทันสมัย และบริหารจัดการได้อย่างมีประสิทธิภาพ

**💡 เหตุผลที่เลือกทำโปรเจกต์นี้**

- ลดความล่าช้าของระบบเดิม: เปลี่ยนการจัดการแบบใช้กระดาษและแรงงานคน (Manual) ที่เสี่ยงต่อข้อมูลตกหล่น มาเป็นระบบดิจิทัลอัตโนมัติ
- ยกระดับความปลอดภัย: นำกล้อง AI และสมองกลฝังตัวมาช่วยเฝ้าระวังพื้นที่เสี่ยงและแจ้งเตือนภัยเชิงรุกแบบ Real-time แทนการดูภาพย้อนหลัง
- ฝึกทักษะวิชาชีพขั้นสูง: บูรณาการความรู้ด้าน Full-stack Web, AI และ IoT มาสร้างชิ้นงานที่ตอบโจทย์ความต้องการของชุมชนจริง
